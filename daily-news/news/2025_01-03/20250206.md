#### ハッカー、Microsoft ADFSのログインページを偽装して資格情報を窃取
https://www.bleepingcomputer.com/news/security/hackers-spoof-microsoft-adfs-login-pages-to-steal-credentials/

- 要約
    - ハッカーが、Microsoft Active Directory Federation Services（ADFS）のログインページを偽装し、資格情報を窃取するフィッシングキャンペーンを展開しています。
    - 主な標的は教育、医療、政府機関で、少なくとも150のターゲットが攻撃を受けています。
    - 攻撃者は、企業のITチームを装ったメールを送り、セキュリティ設定の更新や新しいポリシーの受け入れを促します。
    - 被害者が偽のログインページでユーザー名、パスワード、MFAコードなどの資格情報を入力すると、攻撃者はこれを使って被害者のアカウントにログインします。
    - この手法により、攻撃者は多要素認証（MFA）を回避し、組織のシステムに不正アクセスします。
    - 窃取したアカウントを使い、組織内の他の被害者へのメール送信やビジネスメール詐欺（BEC）などの金銭的攻撃を行います。

    - 不審なログインページに遭遇した場合、直ちにIT部門に報告してください。

#### AMD、ハッカーが悪意のあるマイクロコードパッチをロードできるバグを修正
https://www.bleepingcomputer.com/news/security/amd-fixes-bug-that-lets-hackers-load-malicious-microcode-patches/

- 要約
    - AMDは、CVE-2024-56161として追跡される高深刻度の脆弱性を修正するため、緩和策とファームウェアの更新をリリースしました。
    - この脆弱性は、CPU ROMマイクロコードパッチローダーの不適切な署名検証に起因します。
    - ローカル管理者権限を持つ攻撃者がこの脆弱性を悪用すると、AMDのSecure Encrypted Virtualization-Secure Nested Paging（SEV-SNP）下で実行される機密ゲストの機密性と完全性が損なわれる可能性があります。
    - AMDは、影響を受けるすべてのプラットフォームに対して、悪意のあるマイクロコードの実行を防ぐためのマイクロコード更新を提供しています。
    - 一部のプラットフォームでは、SEV-SNPの認証のためにSEVファームウェアの更新が必要であり、ユーザーはシステムBIOSを更新し、再起動して緩和策の認証を有効にする必要があります。

#### CISA、Microsoft .NETおよびApache OFBizの脆弱性を攻撃で悪用されていると指摘
https://www.bleepingcomputer.com/news/security/cisa-tags-microsoft-net-and-apache-ofbiz-bugs-as-exploited-in-attacks/

- 要約
    - CISAは、Microsoft .NET FrameworkおよびApache OFBizに存在する4つの脆弱性を「既知の悪用されている脆弱性」カタログに追加しました。
        - Microsoft .NET Frameworkの脆弱性（CVE-2024-29059）
            - CODE WHITEによって発見された、深刻度が高い（CVSS v3スコア：7.5）情報漏洩のバグ
            - 内部オブジェクトのURIをリークするために悪用される可能性があり、.NET Remoting攻撃を実行するために使用される可能性
            - 2025年1月のセキュリティ更新で修正
        - Apache OFBizの脆弱性（CVE-2024-45195）
            - Rapid7によって発見された、深刻度が критическая（CVSS v3スコア：9.8）のリモートコード実行の脆弱性
            - 認証されていないダイレクトリクエスト攻撃に対して制限されたパスを公開する、強制ブラウジングの脆弱性が原因
            - バージョン18.12.16より前のOFBizに影響
            - 2021年のセキュリティ更新で修正
        - Paessler PRTGネットワーク監視ソフトウェアの脆弱性（CVE-2018-9276およびCVE-2018-19410）
            - CVE-2018-9276はOSコマンドインジェクションの問題
            - CVE-2018-19410はローカルファイルインクルージョンの脆弱性
    - これらの脆弱性は、情報漏洩やリモートコード実行を引き起こす可能性があり、攻撃者によって積極的に悪用されています。
    - CISAは、連邦機関および大規模組織に対し、これらの脆弱性に対するセキュリティ更新を迅速に適用するよう促しています。
