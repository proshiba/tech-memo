#### WhatsAppがプレティーン向けに保護者管理アカウントを導入
https://www.bleepingcomputer.com/news/security/whatsapp-introduces-parent-managed-accounts-for-pre-teens/

- 要約
    - WhatsAppはプレティーン向けの保護者管理アカウントの展開を開始し、保護者が連絡相手や参加可能グループを管理できるようにした。
    - 管理対象アカウントはメッセージと通話のみに制限され、Meta AI、チャンネル、ステータス、位置情報共有は利用できない。
    - 会話はエンドツーエンド暗号化されたままで、保護者を含む第三者は本文や通話内容を閲覧・傍聴できず、設定変更には親PINが必要となる。
    - 初期設定には親子の端末が同時に必要で、保護者が子どもの電話番号と年齢を確認し、子どもの端末のQRコードを読み取って連携する。
    - 既定では子どもの連絡先に保存済みの相手のみが連絡でき、未知の相手には共通グループや国情報付きの案内が表示され、保護者には各種通知が届く。

#### Elementor Ally プラグインの SQLi 脆弱性が25万超の WordPress サイトに影響
https://www.bleepingcomputer.com/news/security/sqli-flaw-in-elementor-ally-plugin-impacts-250k-plus-wordpress-sites/

- 要約
    - Elementor の WordPress 向け Ally プラグインに、認証なしで悪用可能な SQL インジェクション脆弱性 CVE-2026-2313 が見つかった。
    - 影響は Ally 4.0.3 以前に及び、URL パラメータ処理の不備により、攻撃者は URL パス経由で SQL クエリを注入し機密情報を取得できる。
    - Wordfence によると、この不具合は Elementor アカウントへ接続済みで、かつ Remediation モジュールが有効な場合に悪用可能である。
    - ベンダーは 2026年2月23日公開の 4.1.0 で修正したが、WordPress.org の統計では約36%しか更新しておらず、25万超のサイトが未対策とされる。
    - 記事では、Ally を 4.1.0 へ更新することに加え、前日に公開された WordPress 6.9.2 の最新セキュリティ更新も直ちに適用するよう勧告している。

    - Ally プラグインをバージョン 4.1.0 に更新する。

#### CISA、攻撃で悪用されている n8n の RCE 脆弱性へのパッチ適用を連邦機関に命令
https://www.bleepingcomputer.com/news/security/cisa-orders-feds-to-patch-n8n-rce-flaw-exploited-in-attacks/

- 要約
    - CISA は 2026年3月11日、攻撃で既に悪用されている n8n の脆弱性 CVE-2025-68613 について、連邦機関に対し対処を命じた。
    - この不具合はワークフロー式評価機能における動的コード資源の制御不備で、認証済み攻撃者が n8n プロセス権限で任意コードを実行できる。
    - 成功すると、機密データへの不正アクセス、ワークフロー改変、システムレベル操作まで可能になり、n8n が保持する各種秘密情報が危険にさらされる。
    - n8n チームは 2025年12月に v1.122.0 で修正済みだが、Shadowserver はインターネット上で 4万超の未修正公開インスタンスを追跡している。
    - CISA は本件を KEV カタログへ追加し、FCEB 機関に 2026年3月25日までの対処を要求するとともに、全ての防御担当者へ早急な保護を促した。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - n8n を v1.122.0 へ更新し、パッチを直ちに適用する。
    - 直ちに更新できない場合は、ワークフローの作成・編集権限を完全に信頼できる利用者のみに制限する。
    - OS 権限とネットワークアクセスを制限し、ベンダーの緩和策に従う。緩和策がない場合は製品利用停止も検討する。

#### 新たな PhantomRaven の npm 攻撃波、88個のパッケージ経由で開発者データを窃取
https://www.bleepingcomputer.com/news/security/new-phantomraven-npm-attack-wave-steals-dev-data-via-88-packages/

- 要約
    - Endor Labs は 2025年11月〜2026年2月の新たな3波を確認し、PhantomRaven が 50 の使い捨てアカウントで 88 個の悪性 npm パッケージを配布したと報告した。
    - 攻撃者は Babel や GraphQL Codegen などの著名プロジェクトに似せる「slopsquatting」を多用し、LLM が提案しそうな名称で不正パッケージを公開していた。
    - PhantomRaven は package.json に外部 URL 依存を記載する Remote Dynamic Dependencies を使い、npm install 時に攻撃者サーバーからマルウェアを取得・実行させる。
    - マルウェアは .gitconfig、.npmrc、環境変数からメールや GitHub、GitLab、Jenkins、CircleCI の CI/CD トークンを収集し、端末の IP や OS、Node 版も取得する。
    - 窃取データは主に HTTP GET で C2 に送信され、POST や WebSocket も使われる。観測された 4 波で基盤はほぼ共通で、81 個の悪性パッケージがなお公開状態にある。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - 依存コンポーネントの正当性を検証する。
    - 信頼できる公開元のパッケージのみ利用する。
    - AI チャットボットや未検証ソースの提案をそのままコピーして使わない。
- その他
    - この攻撃キャンペーンは、Koi Securityによって2025年10月に報告されており、今回はそれが3つの波となって追加で行われている活動の報告である
- ChatGPTの推奨事項
    - 開発環境と CI/CD で最近導入した npm 依存関係を直ちに棚卸しし、不審パッケージの削除と流出の恐れがあるトークンの即時ローテーションを実施してください。

#### Meta が WhatsApp、Facebook、Messenger に新たな詐欺対策ツールを追加
https://www.bleepingcomputer.com/news/security/meta-adds-new-whatsapp-facebook-and-messenger-anti-scam-tools/

#### 人事部門を標的とする新たな「BlackSanta」EDRキラーが確認される
https://www.bleepingcomputer.com/news/security/new-blacksanta-edr-killer-spotted-targeting-hr-departments/

- 要約
    - ロシア語話者の脅威アクターが1年以上にわたり人事部門を狙い、新たなEDRキラー「BlackSanta」を配布するマルウェア攻撃を継続していた。
    - 研究者は、履歴書に見せかけたISOをクラウドストレージ経由で配布するスピアフィッシングが起点とみており、ISO内にはPDF偽装LNK、PowerShell、画像、ICOが含まれていた。
    - LNKから起動されたPowerShellは画像内の隠しデータを復号してメモリ実行し、正規のSumatraPDFと悪性DWrite.dllを使うDLLサイドローディングで次段階を展開する。
    - マルウェアは端末情報をC2へ送信し、サンドボックスや仮想環境の検知、Defender設定の弱体化、追加ペイロード取得、プロセスホロウイングを実行する。
    - BlackSantaはDefender除外や通知抑止に加え、BYODドライバーを悪用してAV/EDR/SIEM/フォレンジック系プロセスをカーネルレベルで停止させる。

#### 新たな BeatBanker Android マルウェアが Starlink アプリを装って端末を乗っ取る
https://www.bleepingcomputer.com/news/security/new-beatbanker-android-malware-poses-as-starlink-app-to-hijack-devices/

- 要約
    - 新たに見つかった Android マルウェア「BeatBanker」は、公式 Google Play ストアを装う偽サイト上で Starlink アプリを名乗り、利用者に導入を促す。
    - このマルウェアは銀行型トロイの木馬機能と Monero マイニング機能を備え、認証情報の窃取や暗号資産取引の改ざんも可能とされる。
    - Kaspersky はブラジルの利用者を狙う攻撃で BeatBanker を確認し、最新版では銀行機能の代わりに BTMOB RAT を展開して端末を遠隔操作する。
    - BeatBanker はネイティブライブラリで隠された DEX を復号してメモリ上で読み込み、偽の Play ストア更新画面で追加ペイロード導入権限をだまし取る。
    - 永続化には output8.mp3 によるほぼ聞こえない音声の連続再生を使い、端末状態を監視しながら採掘を制御して発見を回避する。
- IOCの列挙

- 推奨事項
    - 信頼できる公開元・配布元であると確認できない限り、公式 Google Play ストア以外から APK をサイドロードしない。
    - アプリの機能に無関係な危険な権限が付与されていないか確認する。
    - 定期的に Play Protect のスキャンを実行する。
- その他
    - なし
- ChatGPTの推奨事項
