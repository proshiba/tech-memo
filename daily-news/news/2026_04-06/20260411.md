#### 米国の産業用機器約4,000台がイラン系サイバー攻撃にさらされている
https://www.bleepingcomputer.com/news/security/nearly-4-000-us-industrial-devices-exposed-to-iranian-cyberattacks/

- 要約
    - 米連邦機関の合同警告によると、イラン系国家支援ハッカーは2026年3月以降、米重要インフラのRockwell/Allen-Bradley製PLCを標的としている。
    - 攻撃では装置の project file が抜き取られ、HMI や SCADA 表示のデータ改ざんが確認され、運用妨害と金銭的損失が発生した。
    - Censys は EtherNet/IP に応答する Rockwell/Allen-Bradley 機器を世界で5,219台確認し、その74.6%に当たる3,891台が米国にあると報告した。
    - 米国内の露出機器は携帯回線 ASN に偏っており、現場設置機器がセルラーモデム経由で直接公開されている可能性が示された。
    - 記事は、公開PLCの防御強化としてFW保護やインターネット切り離し、ログ点検、OTポート監視、MFA、更新適用を挙げている。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - PLC をファイアウォールで保護するか、インターネットから切り離す。
    - ログを確認して不正活動の痕跡を探し、特に海外ホスティング事業者起点の OT ポート通信を警戒する。
    - OT ネットワークへの MFA を強制し、PLC を最新状態に保ち、未使用のサービスと認証方式を無効化する。
- その他
    - 攻撃者はイラン系の国家支援ハッカー集団と説明され、米政府機関は最近の活動活発化がイランと米国・イスラエル間の敵対行為への反応である可能性を示している。
- ChatGPTの推奨事項
    - 直ちにインターネット公開中の Rockwell/Allen-Bradley PLC を特定し、外部公開停止または FW 制限、OT ログ点検、海外ホスティング起点通信の確認を実施してください。

#### CPUIDが侵害され、CPU-ZとHWMonitorのダウンロード経由でマルウェアを配布
https://www.bleepingcomputer.com/news/security/supply-chain-attack-at-cpuid-pushes-malware-with-cpu-z-hwmonitor/

    - 記事掲載時点で問題は修正済みで、CPU-Z と HWMonitor のクリーン版が再び配布されている一方、影響版数や利用者向け対処は調査中だった。

#### Microsoft：カナダの従業員が「Payroll Pirate」攻撃の標的に
https://www.bleepingcomputer.com/news/microsoft/microsoft-canadian-employees-targeted-in-payroll-pirate-attacks/

- 要約
    - Microsoft は、Storm-2755 と追跡する金銭目的の脅威アクターが、カナダの従業員アカウントを乗っ取り給与振込先を奪う「Payroll Pirate」攻撃を行っていると報告した。
    - 攻撃者は、Microsoft 365 のサインイン画面を装う悪性ページへ検索広告や SEO 汚染で誘導し、認証トークンとセッション Cookie を盗んで AiTM 攻撃を成立させる。
    - 盗んだセッショントークンの再利用により、再認証なしで MFA を回避し、被害者メールボックスへ侵入して直接預金」または「銀行」といった単語を隠しフォルダへ振り分ける受信トレイルールを作成し、攻撃に気づかれづらいように細工した。
    - その後、攻撃者は「Question about direct deposit」という件名で人事へ連絡し、ソーシャルエンジニアリングが失敗した場合は Workday へ直接ログインして振込先口座を変更する。
    - Microsoft は、防御策としてレガシー認証の遮断とフィッシング耐性 MFA の導入、侵害時のトークン失効、悪性 inbox rule 削除、認証情報の再設定を勧告している。
- IOCの列挙
    - bluegraintours[.]com,FQDN名,知られていない,Microsoft 365 サインイン画面を装う悪性ページのホストとして記載,NA
- 推奨事項
    - レガシー認証プロトコルをブロックし、フィッシング耐性のある MFA を実装する。
    - 侵害の兆候が見つかった場合は、侵害されたトークンとセッションを直ちに失効させる。
    - 悪性 inbox rule を削除し、影響を受けたアカウントの MFA 方法と認証情報をリセットする。
- その他
    - 攻撃者は Storm-2755 と追跡される金銭目的の脅威アクターで、給与振込先の乗っ取りを目的とした BEC 系手口の一種として説明されている。
- ChatGPTの推奨事項
    - Microsoft 365 と Workday を利用する組織は、直ちに direct deposit 関連メールの inbox rule、不審セッション、振込先変更履歴を点検し、フィッシング耐性 MFA を適用してください。

#### Google、モバイル端末向けにGmailのエンドツーエンド暗号化を展開
https://www.bleepingcomputer.com/news/google/google-rolls-out-gmail-end-to-end-encryption-on-mobile-devices/

- 要約
    - Google は Gmail のエンドツーエンド暗号化(E2EE)を Android / iOS の Gmail アプリへ拡大し、追加ツールなしで作成・閲覧できるようにした。
    - Gmail アプリ利用者には暗号化メールが通常の受信メールとして届き、他メールサービス利用者もブラウザー経由で閲覧できる。
    - 利用対象は Enterprise Plusライセンスを持ち、Assured Controls / Assured Controls Plus を持つ CSE 利用組織で、管理者による端末有効化が必要である。
    - この機能は Client-side Encryption に基づき、暗号鍵を Google 外で組織が管理することで、Google や第三者が内容を読めない設計となっている。
    - 送信時は作成画面のロックアイコンから「Additional encryption」を有効化して、エンドツーエンド暗号化メッセージを送る。

https://www.bleepingcomputer.com/news/security/new-venom-phishing-attacks-steal-senior-executives-microsoft-logins/

- 要約
    - 未公表のPhaaS「VENOM」を使う攻撃者が、複数業界のC-suite幹部を狙い、少なくとも2025年11月以降活動している。
    - メールはMicrosoft SharePointの文書共有通知を装い、偽のメールスレッドやHTMLノイズで社内連絡風に見せる高度な個別化が施されていた。
    - 本文にはUnicodeで描いたQRコードが含まれ、検査ツールを回避して被害者をモバイルへ誘導し、標的のメールアドレスはURLフラグメントで二重Base64化される。
    - 誘導先では研究者やサンドボックスをふるい落とした後、Microsoftログインをリアルタイム中継するAiTMページやデバイスコード承認へ誘導する手口が使われた。
    - いずれの手口でも認証中に永続的アクセスが確立され得るため、記事はMFAのみでは不十分とし、FIDO2や条件付きアクセス強化を勧めている。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - C-suite幹部はFIDO2認証を利用する。
    - 不要な場合はデバイスコードフローを無効化する。
    - トークン悪用を防ぐため、より厳格な条件付きアクセス ポリシーを実装する。
- その他

- ChatGPTの推奨事項

#### GlassWormがネイティブ化：新たなZig製ドロッパーがマシン上の全IDEに感染
https://www.aikido.dev/blog/glassworm-zig-dropper-infects-every-ide-on-your-machine

- 要約
    - Aikido は 2025年3月以降追跡する GlassWorm の新手口として、OpenVSX 上の偽 WakaTime 拡張 code-wakatime-activity-tracker を確認した。
    - この拡張は activate() 実行時に Zig 製ネイティブバイナリを読み込み、JavaScript サンドボックス外で OS 権限の処理を実行する。
    - バイナリは VS Code、Cursor、Windsurf、VSCodium、Positron など VS Code 互換 IDE を列挙し、見つけた全 IDE に横展開する。
    - その後、攻撃者管理の GitHub Releases から steoates.autoimport を装う悪性 .vsix を取得し、各 IDE の CLI を使って密かに導入する。
    - 記事によると第2段階は既知の GlassWorm ドロッパーで、ロシア圏を除外しつつ秘密情報の窃取や永続的 RAT 展開につながる。

- 推奨事項
    - specstudio/code-wakatime-activity-tracker を導入している、または IDE の拡張一覧に floktokbok.autoimport が見つかる場合は、端末を侵害済みとして扱う。
    - その端末からアクセス可能だった秘密情報をローテーションする。
    - Aikido 利用者は central feed を確認し、夜間再スキャンを待たずに手動再スキャンを実施する。
- その他
