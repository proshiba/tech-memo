#### 恐喝グループがRockstar Gamesの盗難分析データを漏えい
https://www.bleepingcomputer.com/news/security/stolen-rockstar-games-analytics-data-leaked-by-extortion-gang/

- 要約
    - Rockstar Games は、Anodot の最近のセキュリティ事故に関連するデータ侵害を受け、ShinyHunters が流出データを恐喝サイトで公開したと報じられた。
    - 攻撃者は、Anodot 事故で盗んだ認証トークンを使って Snowflake 環境へ侵入し、Rockstar のデータを取得したと主張している。

    - 本件は、Anodot を利用していた複数企業に対するより大きなデータ窃取キャンペーンの一部で、Snowflake も影響顧客アカウントをロックして通知したとされる。

#### wolfSSLライブラリの重大な欠陥により偽造証明書の利用が可能に
https://www.bleepingcomputer.com/news/security/critical-flaw-in-wolfssl-library-enables-forged-certificate-use/

- 要約
    - wolfSSL に重大な暗号検証不備 CVE-2026-5194 が見つかり、ECDSA 署名検証時のハッシュアルゴリズムやサイズ確認不足で弱いダイジェストが受理され得る。
    - この欠陥により、脆弱な wolfSSL を使う機器やアプリが、悪意あるサーバーや接続向けの偽造証明書を正当なものとして受け入れる可能性がある。
    - 影響は ECDSA/ECC、DSA、ML-DSA、Ed25519、Ed448 など複数アルゴリズムに及び、ECC と EdDSA または ML-DSA を有効化した構成では更新が推奨される。
    - wolfSSL は組み込み機器、IoT、産業制御、ルーター、自動車、航空宇宙などで広く使われ、記事では 50 億超のアプリや機器で利用されているとされる。
    - 本件は 2026年4月8日公開の wolfSSL 5.9.1 で修正済みだが、Linux distributionパッケージやベンダーファームウェア利用時は下流ベンダー情報の確認も必要とされた。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - ECC と EdDSA または ML-DSA を有効化したビルドでは、最新の wolfSSL リリースへ更新する。
    - Linux 配布パッケージ、ベンダーファームウェア、組み込み SDK を利用している場合は、下流ベンダーのアドバイザリを確認する。
    - wolfSSL を利用する組織は、自社展開状況を確認し、速やかにセキュリティ更新を適用する。
- その他
    - なし
- ChatGPTの推奨事項
    - wolfSSL 利用有無と版数を直ちに棚卸しし、影響環境は 5.9.1 以降またはベンダー修正版へ緊急更新してください。

#### OpenAI、Axios攻撃がコード署名ワークフローに及んだためmacOS証明書をローテーション
https://www.bleepingcomputer.com/news/security/openai-rotates-macos-certs-after-axios-attack-hit-code-signing-workflow/

- 要約
    - OpenAI は、2026年3月31日に GitHub Actions ワークフローが改ざん版 Axios 1.14.1 を実行したことを受け、macOS 用コード署名証明書のローテーションを開始した。
    - 当該ワークフローは ChatGPT Desktop、Codex、Codex CLI、Atlas などの macOS アプリ署名用証明書へアクセスできたため、同社は証明書流出の証拠がなくても予防的に失効対応を進めている。
    - OpenAI は外部調査会社と調査を行い、証明書露出や悪性ソフト署名への悪用、ユーザーデータ侵害、知的財産侵害、ソフト改ざんの証拠は見つからなかったとしている。
    - ただし旧証明書が攻撃者に渡っていれば、OpenAI 正規署名に見える macOS アプリを作られる恐れがあるため、Apple と連携して旧証明書での今後の notarization を防ぐとしている。
    - 影響は macOS アプリに限定され、Web、iOS、Android、Windows、Linux、ならびにユーザーアカウント、パスワード、API キーには影響せず、旧版は2026年5月8日以降に動作停止し得る。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - macOS 利用者は OpenAI アプリを最新バージョンへ更新する。
    - 更新はアプリ内機能または公式ダウンロードページから行う。
    - メール、広告、第三者サイトのリンクからソフトをインストールしない。
- その他
    - 攻撃者は Axios 事案に関連する北朝鮮系脅威アクター UNC1069 と結び付けられており、ソーシャルエンジニアリングでメンテナーをだまして悪性版を公開したと記載されている。
- ChatGPTの推奨事項
    - OpenAI の macOS アプリを使っている場合は、正規配布元から最新版へ直ちに更新し、古い版の継続利用をやめてください。

#### Booking.comの新たなデータ侵害で予約PINのリセットを実施
https://www.bleepingcomputer.com/news/security/new-bookingcom-data-breach-forces-reservation-pin-resets/

- 要約
    - Booking.com は、一部の予約情報に紐づく利用者データへ第三者が不正アクセスしたことを認め、影響利用者へのメール通知を開始した。
    - 漏えいした可能性があるのは氏名、メールアドレス、郵送先住所、電話番号、宿泊施設とのやり取り内容で、既存・過去予約の PIN が更新された。
    - すでにSNS等では、流出した予約情報を悪用したと思われる詐欺の報告も上がっている。
    - 通知では、不審なメールや電話に注意し、Booking.com は機微情報や銀行振込を求めないと案内し、疑わしいリンクは開かないよう促した。
    - 一方で、通知を受けた一部利用者はアプリ内で同様の警告を受けておらず、正当な連絡かどうか混乱が生じたとも記事は伝えている。
    - Booking.com は影響人数を公表していないが、問題を封じ込めたとしており、影響者には個別通知と多言語の24時間サポートを提供するとした。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - 不審なメールや電話に注意する。
    - Booking.com は機微情報や銀行振込を求めないため、そのような要求には応じない。
    - 宿泊施設や Booking.com を装っているように見えるメールでも、リンクをクリックしない。
- その他
    - なし
- ChatGPTの推奨事項
    - Booking.com を利用している場合は、予約番号に紐づく PIN の変更有無を直ちに確認し、不審な連絡のリンクや送金依頼には応じないでください。

#### Adobe、Acrobat/Readerのゼロデイ脆弱性に対する緊急修正を公開
https://www.bleepingcomputer.com/news/security/adobe-rolls-out-emergency-fix-for-acrobat-reader-zero-day-flaw/

- 要約
    - Adobe は、少なくとも12月以降のゼロデイ攻撃で悪用されていた Acrobat / Reader の脆弱性 CVE-2026-34621 向け緊急アップデートを公開した。
    - 細工された PDF を開くだけでサンドボックス制限を回避し、権限付き JavaScript API を呼び出して任意ファイルの読取や追加コード取得が可能になる。
    - 攻撃では util.readFileIntoStream() でローカルファイルを読み、RSS.addFeed() でデータ送信や追加コード取得を行う手口が使われたと説明されている。
    - 影響を受けるのは Acrobat DC / Reader DC 26.001.21367 以下と Acrobat 2024 24.001.30356 以下で、修正版が Windows と macOS 向けに公開された。
    - Adobe は回避策を示しておらず、更新適用が唯一の推奨対応であり、不審な PDF は送信元未確認のまま開かないよう注意を促している。
- IOCの列挙
    - IOC情報なし
- 推奨事項
    - Help > Check for Updates から自動更新を実行する。
    - 必要に応じて Adobe の公式ソフトウェアポータルからインストーラーを入手する。
    - 不審または未承諾の送信元から届いた PDF は警戒し、疑わしい場合はサンドボックス環境で開く。
- その他
    - なし
- ChatGPTの推奨事項
    - Acrobat / Reader の版数を直ちに確認し、対象版は緊急更新し、更新完了まで未承諾の PDF の開封を停止してください。

https://www.malwarebytes.com/blog/scams/2026/04/fake-claude-site-installs-malware-that-gives-attackers-access-to-your-computer

- 要約
    - Malwarebytes は、Anthropic の Claude を装う偽サイトを確認し、見た目は通常の Claude が動く一方で裏で PlugX 連鎖を展開する ZIP を配布していた。

    - ドロッパーは `NOVUpdate.exe`、`avk.dll`、`NOVUpdate.exe.dat` を Startup に配置し、署名済み G DATA 更新プログラムを使う DLL サイドローディングで実行を隠す。
    - サンドボックスでは `WScript.exe` による`NOVUpdate.exe`配置後、22 秒で `8[.]217[.]190[.]58:443` への外向き通信が始まり、複数回の C2 コールバックが確認された。
    - 記事は、最近文書化された手口が AI 人気に便乗して再利用された例とし、見た目が正常でも背後で遠隔操作型マルウェアが常駐すると警告している。
- IOCの列挙
    - 8[.]217[.]190[.]58,IPアドレス,知られていない,`NOVUpdate[.]exe` の初回外向き TCP 接続先として観測された PlugX の C2 宛先,グローバル
- 推奨事項
    - Startup フォルダに `NOVUpdate[.]exe`、`avk[.]dll`、`NOVUpdate[.]exe[.]dat` がないか確認し、`C:\Program Files (x86)\Anthropic\Claude\Cluade\` の誤記ディレクトリも点検する。
    - 見つかった場合は直ちにインターネットから切断し、完全スキャンを実行し、`8[.]217[.]190[.]58` への外向き通信を FW / プロキシログで確認する。
    - Claude は `claude[.]com/download` の公式経路からのみ入手し、メール、広告、公式外の「Pro」版リンクは開かない。
- その他
